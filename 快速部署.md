# 🚀 业主大会投票小程序 - 一键部署命令

**只需 3 步，快速启动后端服务！**

---

## 📋 第 1 步：安装环境依赖

以**管理员身份**打开 PowerShell，运行：

```powershell
# 安装 Chocolatey（如果已安装可跳过）
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# 安装 PostgreSQL 和 Redis
choco install postgresql14 redis-64 -y
```

**重启终端**后继续下一步。

---

## 📋 第 2 步：启动服务

以**普通用户身份**打开 PowerShell，运行：

```powershell
# 进入项目目录
cd C:\Users\Admin\.openclaw\workspace\agents\engineer\projects\业主大会投票小程序

# 启动数据库和缓存服务
net start postgresql-x64-14
redis-server --service-start

# 创建数据库
psql -U postgres -c "CREATE DATABASE vote_db;"

# 进入后端目录
cd backend

# 创建并激活虚拟环境
python -m venv venv
.\venv\Scripts\Activate.ps1

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
Copy-Item .env.example .env

# 提示：编辑 .env 文件，填入 PostgreSQL 密码
notepad .env

# 运行数据库迁移
alembic upgrade head

# 启动后端服务
python main.py
```

## 🎯 常用命令速查

### 服务管理
```powershell
# 启动 PostgreSQL
net start postgresql-x64-14

# 启动 Redis
redis-server --service-start

# 停止 PostgreSQL
net stop postgresql-x64-14

# 停止 Redis
redis-server --service-stop
```

### 后端服务
```powershell
# 进入项目目录
cd C:\Users\Admin\.openclaw\workspace\agents\engineer\projects\业主大会投票小程序

# 激活虚拟环境
cd backend
.\venv\Scripts\Activate.ps1

# 启动服务
python main.py

# 运行迁移
alembic upgrade head

# 创建管理员
python scripts/create_admin.py
```

### 数据库操作
```powershell
# 连接数据库
psql -U postgres -d vote_db

# 备份数据库
pg_dump -U postgres vote_db > backup.sql

# 恢复数据库
psql -U postgres vote_db < backup.sql
```

### Redis 操作
```powershell
# 连接 Redis
redis-cli

# 测试连接
redis-cli ping
# 返回 PONG 表示正常

# 查看所有键
redis-cli KEYS "*"

# 清空所有数据（慎用！）
redis-cli FLUSHALL
```

## 🔍 故障排查

### 问题 1：`psql` 命令不存在
**解决：** 使用完整路径
```powershell
& "C:\Program Files\PostgreSQL\14\bin\psql.exe" -U postgres -c "CREATE DATABASE vote_db;"
```

### 问题 2：端口被占用
**解决：** 查找并结束占用进程
```powershell
# 查找占用 8000 端口的进程
netstat -ano | findstr :8000

# 结束进程
taskkill /F /PID 进程号
```

### 问题 3：虚拟环境激活失败
**解决：** 修改执行策略
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.\venv\Scripts\Activate.ps1
```

### 问题 4：pip 安装依赖失败
**解决：** 使用国内镜像
```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
