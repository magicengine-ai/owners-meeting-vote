# 微信小程序配置指南

## 📋 部署成功后需要修改的配置

微信云托管部署成功后，需要修改小程序前端的服务器域名配置。

---

## 🔧 第一步：获取云托管服务域名

1. 登录 [微信云托管控制台](https://mp.weixin.qq.com/)
2. 进入你的小程序 → 云托管
3. 选择环境：`prod-3gxp2c5a6c9a177f`
4. 点击 **服务管理** → 选择 `vote-api` 服务
5. 在服务详情页找到 **服务域名**，格式如下：
   ```
   https://<服务 ID>-<环境 ID>.service.tcloudbase.com
   ```
   
   例如：`https://vote-api-prod-3gxp2c5a6c9a177f.service.tcloudbase.com`

---

## 🔧 第二步：修改小程序代码配置

需要修改以下 3 个文件中的 `baseUrl`：

### 1. `frontend/app.js`
### 2. `frontend/utils/request.js`
### 3. `frontend/utils/config.js`

这 3 个文件已经更新为占位符：`https://YOUR-CLOUDRUN-DOMAIN.service.tcloudbase.com`

请将 `YOUR-CLOUDRUN-DOMAIN` 替换为你的实际服务域名。

---

## 🔧 第三步：配置小程序服务器域名（重要！）

在微信公众平台配置合法域名，否则小程序无法请求：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入小程序管理后台
3. 左侧菜单：**开发** → **开发管理** → **开发设置**
4. 找到 **服务器域名** 配置
5. 添加你的云托管服务域名到：
   - ✅ **request 合法域名**
   - ✅ **uploadFile 合法域名**
   - ✅ **downloadFile 合法域名**
