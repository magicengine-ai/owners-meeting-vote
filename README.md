## 🎯 项目目标

开发一款基于微信小程序的业主大会投票系统，实现：
- ✅ 业主身份认证（房产证 OCR + 政府系统验证）
- ✅ 在线投票（支持赞成/反对/弃权）
- ✅ 会议管理（创建、通知、签到、记录）
- ✅ 区块链存证（投票和会议记录不可篡改）
- ✅ 消息推送（小程序消息 + 短信通知）

---

## 🏗️ 技术架构

### 后端技术栈
- **运行时：** Node.js 18+ / Python 3.9+
- **框架：** FastAPI / Express
- **数据库：** PostgreSQL + Redis
- **区块链：** 百度超级链 / 蚂蚁链
- **OCR：** 百度 AI / 腾讯云 OCR
- **部署：** Docker + Kubernetes

### 前端技术栈
- **平台：** 微信小程序
- **框架：** Taro / 原生小程序
- **UI 库：** Vant Weapp / Taro UI
- **状态管理：** Vuex / Redux

## 🔧 开发环境

### 环境要求
- Node.js >= 18.0.0
- Python >= 3.9
- PostgreSQL >= 14
- Redis >= 6.0
- Docker >= 20.10

### 本地开发
```bash
# 后端
cd backend
pip install -r requirements.txt
python main.py

# 前端
cd frontend
npm install
npm run dev
```
