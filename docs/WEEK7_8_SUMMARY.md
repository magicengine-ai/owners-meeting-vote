## 🚀 会议流程

### 创建会议
```
1. 管理员创建会议
   ↓
2. 设置时间、地点、议程
   ↓
3. 发布会议通知
```

### 参加会议
```
1. 用户查看会议列表
   ↓
2. 报名参加会议
   ↓
3. 会议当天签到
   ↓
4. 查看会议材料
```

### 会议统计
```
1. 报名人数统计
   ↓
2. 签到率统计
   ↓
3. 参会人员导出
```

## ⚠️ 注意事项

### 1. 会议权限
- 只有管理员才能创建会议
- 所有认证用户可以报名
- 报名后才能签到

### 2. 会议状态
- upcoming: 待开始
- ongoing: 进行中
- completed: 已结束
- cancelled: 已取消

### 3. 签到时间
- 会议开始前可以签到
- 会议进行中可以签到
- 会议结束后不能签到
