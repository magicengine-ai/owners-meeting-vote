## 🚀 快速开始

### 1. 克隆项目

```bash
git clone git@github.com:magicengine-ai/owners-meeting-vote.git
cd owners-meeting-vote
```

### 2. 启动后端

```bash
cd backend
copy .env.example .env
# 编辑 .env 配置

pip install -r requirements.txt
python main.py
```

访问：http://localhost:8000/docs

### 3. 打开前端

1. 打开微信开发者工具
2. 导入 `frontend` 目录
3. 配置 AppID（或使用测试号）
4. 编译运行

- **框架：** FastAPI 0.109
