#### 存证数据

```python
    "user_id": 100,
    "options": ["赞成"],
    "timestamp": "2026-03-12T23:00:00"
}
```

#### 哈希生成
```python
def generate_chain_hash(vote_record):
    data = {
        "vote_id": vote_record.vote_id,
        "user_id": vote_record.user_id,
        "options": vote_record.options,
        "timestamp": vote_record.created_at.isoformat()
    }
    return hashlib.sha256(json.dumps(data).encode()).hexdigest()
```

## ⚠️ 注意事项

### 1. 投票权限
- 只有认证业主才能投票
- 管理员/业委会才能创建投票
- 一人一票（按房产证）

### 2. 投票时间
- 未到开始时间不能投票
- 超过结束时间不能投票
- 时间验证在后端

### 3. 防止作弊
- 防止重复投票
- 区块链存证
- 投票记录不可篡改

### 4. 隐私保护
- 投票内容保密
- 只显示统计结果
- 不公开个人投票
