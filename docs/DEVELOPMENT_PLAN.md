## 🔧 技术选型

### 后端
- **框架：** FastAPI
- **数据库：** PostgreSQL 14
- **缓存：** Redis 6
- **认证：** JWT
- **区块链：** 百度超级链
- **OCR：** 百度 AI
- **部署：** Docker + K8s

### 前端
- **平台：** 微信小程序
- **UI 库：** Vant Weapp
- **构建工具：** 小程序开发者工具

### DevOps
- **CI/CD：** GitHub Actions
- **监控：** Prometheus + Grafana
- **日志：** ELK Stack

### 技术风险
1. **政府 API 对接困难**
   - 应对：提前调研，准备备选方案

   - 应对：异步上链，批量处理

3. **高并发投票**
   - 应对：Redis 缓存，消息队列

1. **需求变更**
   - 应对：敏捷开发，快速迭代

2. **人员不足**
   - 应对：优先开发核心功能
