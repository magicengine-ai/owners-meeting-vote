## 🚀 完整审核流程

### 用户提交认证
```
1. 用户上传房产证
   ↓
2. OCR 自动识别
   ↓
3. 确认提交
   ↓
4. 状态变为"审核中"
   ↓
5. 等待管理员审核
```

### 管理员审核
```
1. 进入审核列表
   ↓
2. 查看用户信息
   ↓
3. 选择通过/拒绝
   ↓
4. 拒绝需填写原因
   ↓
5. 系统自动发送通知
   ↓
6. 记录审核历史
```

### 用户收到通知
```
审核通过：
- 微信模板消息
- 个人中心显示"已认证"
- 可以参与投票

审核拒绝：
- 微信模板消息（含原因）
- 个人中心显示"认证失败"
- 可以重新认证
```
