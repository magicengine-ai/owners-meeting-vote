## 🏗️ 技术架构

### 后端技术栈
- **框架：** FastAPI 0.109
- **数据库：** PostgreSQL 14 + SQLAlchemy 2.0
- **缓存：** Redis 6
- **认证：** JWT (PyJWT 2.8)
- **OCR：** 百度 AI
- **区块链：** 百度超级链
- **消息：** 微信模板消息

### 前端技术栈
- **平台：** 微信小程序
- **语言：** 原生 WXML/WXSS/JavaScript
- **UI：** 自定义组件
- **网络：** 封装 request

### DevOps
- **容器：** Docker + Docker Compose
- **部署：** Nginx 反向代理
- **CI/CD：** GitHub Actions

## 🚀 快速开始

### 开发环境
```bash
# 1. 克隆代码
git clone git@github.com:magicengine-ai/owners-meeting-vote.git

# 2. 安装依赖
cd owners-meeting-vote/backend
pip install -r requirements.txt

# 3. 配置环境
cp .env.example .env
vim .env

# 4. 启动服务
python main.py

# 5. 访问文档
http://localhost:8000/docs
```

### 生产环境
```bash
# 1. 部署
./scripts/deploy.sh

# 2. 查看日志
docker-compose logs -f

# 3. 健康检查
curl http://localhost:8000/health
```
