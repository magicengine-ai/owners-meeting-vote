## 🎯 核心改动

### 1. 端口适配
```python
# 本地开发：8000
# 云托管：8080（微信官方推荐）
PORT: int = int(os.getenv("PORT", "8080"))
```

### 2. 数据库自动检测
```python
# 自动检测环境
if os.getenv("TENCENTCLOUD_MYSQL_HOST"):
    DB_TYPE = "mysql"  # 云托管
else:
    DB_TYPE = "postgresql"  # 本地开发
```

### 3. 微信原生鉴权
```python
# 从请求头获取 OpenID（云托管自动注入）
openid = request.headers.get("X-WX-OPENID")

# 依赖注入
@router.get("/user/profile")
async def get_profile(current_user: Dict = Depends(get_current_user)):
    return {"openid": current_user["openid"]}
```

### 4. 免鉴权调用微信接口
```python
# 无需 access_token！
await WeChatAPI.send_subscribe_message(
    openid="oXXXX",
    template_id="XXXX",
    data={"thing1": {"value": "投票标题"}}
)
```

## 📊 部署方式对比

| 方式 | 优点 | 适用场景 |
|------|------|----------|
| **部署脚本** | 简单快速，一键完成 | 首次部署、本地测试 |
| **GitHub Actions** | 自动化，代码推送即部署 | 团队协作、生产环境 |
| **CloudBase CLI** | 灵活可控，支持回滚 | 手动部署、紧急修复 |

**推荐：** 开发阶段用脚本，生产环境用 GitHub Actions

## 🚀 部署步骤（快速版）

### 1. 开通云托管
```
微信公众平台 -> 开发 -> 云托管 -> 开通
```

### 2. 创建数据库
```
云托管控制台 -> 数据库 -> MySQL -> 创建（2 核 4G）
```

### 3. 运行部署脚本
```powershell
cd 业主大会投票小程序
.\cloud-deploy\deploy-to-cloud.ps1
```

### 4. 配置环境变量
```
云托管控制台 -> 服务详情 -> 环境变量
- JWT_SECRET_KEY=<随机密钥>
- WECHAT_APP_ID=wx8888888888888888
- WECHAT_APP_SECRET=<小程序密钥>
```

### 5. 配置域名
```
微信公众平台 -> 开发 -> 开发设置
-> 服务器域名 -> 添加云托管服务域名
```

### 6. 验证部署
```bash
curl https://<云托管服务域名>/health
# 返回：{"status": "ok", "version": "1.0.0"}
```
