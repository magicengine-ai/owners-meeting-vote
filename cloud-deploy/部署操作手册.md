## 1. 准备工作

- ✅ Git 已安装

| 信息 | 获取位置 | 示例 |

| 小程序 AppID | 微信公众平台 -> 开发 -> 开发设置 | wx8888888888888888 |
| 小程序 AppSecret | 微信公众平台 -> 开发 -> 开发设置 | 1234567890abcdef |
| 百度 OCR AppID | 百度 AI 开放平台 | 12345678 |
| 百度 OCR API Key | 百度 AI 开放平台 | abcdefghijklmnop |

## 2. 开通微信云托管

### 步骤 1：登录微信公众平台

访问：https://mp.weixin.qq.com/

### 步骤 2：进入云托管

左侧菜单：**开发** -> **云托管**

### 步骤 3：开通服务

1. 点击「开通云托管」
2. 选择环境（建议：生产环境）
3. 同意服务协议
4. 点击「开通」

> ⏱️ 开通时间：约 1-2 分钟

---

## 3. 创建数据库

### 3.1 创建 MySQL 数据库

1. 云托管控制台 -> **数据库** -> **MySQL**
2. 点击「创建数据库」
3. 选择配置：
   - **CPU**: 2 核
   - **内存**: 4G
   - **存储**: 100G
4. 设置账号密码（请妥善保管）
5. 点击「创建」

> ⏱️ 创建时间：约 3-5 分钟

### 3.2 创建 Redis（可选）

1. 云托管控制台 -> **数据库** -> **Redis**
2. 点击「创建数据库」
3. 选择配置：
   - **容量**: 1G
4. 设置密码
5. 点击「创建」

## 4. 部署方式一：使用部署脚本（推荐）

### 步骤 1：运行部署脚本

```powershell
cd 业主大会投票小程序
.\cloud-deploy\deploy-to-cloud.ps1
```

### 步骤 2：选择部署方式

脚本会提示选择：
```
请选择部署方式:
  1. Docker 镜像部署（推荐）
  2. CloudBase CLI 部署
  3. 仅构建 Docker 镜像（不部署）
  0. 退出
```

选择 **1** 或 **2**

### 步骤 3：按提示操作

根据脚本提示完成：
- Docker 镜像构建
- 推送到腾讯云镜像仓库
- 云托管服务创建

## 5. 部署方式二：使用 GitHub Actions 自动部署

### 步骤 1：配置 GitHub Secrets

在 GitHub 仓库 -> Settings -> Secrets and variables -> Actions

添加以下 Secrets：

| Secret 名称 | 值 | 说明 |
|------------|-----|------|
| `CLOUDBASE_API_KEY` | xxx | 云托管 API Key |
| `CLOUDBASE_ENV_ID` | xxx | 云托管环境 ID |

### 步骤 2：获取 API Key

1. 访问：https://cloud.weixin.qq.com/
2. 点击右上角头像 -> **访问密钥**
3. 点击「新建密钥」
4. 复制 API Key

### 步骤 3：获取环境 ID

1. 云托管控制台 -> **环境管理**
2. 复制环境 ID（如：env-123456）

### 步骤 4：推送代码触发部署

```bash
git push origin main
```

GitHub Actions 会自动：
1. 构建 Docker 镜像
2. 推送到云托管
3. 部署服务

## 6. 部署方式三：使用 CloudBase CLI 手动部署

### 步骤 1：安装 CLI

```bash
npm install -g @cloudbase/cli
```

### 步骤 2：登录

```bash
cloudbase login
```

扫码登录微信。

### 步骤 3：初始化项目

```bash
cd 业主大会投票小程序
cloudbase init
```

选择已有环境或创建新环境。

### 步骤 4：部署

```bash
cloudbase deploy -f cloudbaserc.json
```

## 7. 配置环境变量

### 7.1 自动注入的变量（无需配置）

创建数据库后，以下变量会自动注入：

```bash
TENCENTCLOUD_MYSQL_HOST
TENCENTCLOUD_MYSQL_PORT
TENCENTCLOUD_MYSQL_DATABASE
TENCENTCLOUD_MYSQL_USERNAME
TENCENTCLOUD_MYSQL_PASSWORD

TENCENTCLOUD_REDIS_HOST      # 如果创建了 Redis
TENCENTCLOUD_REDIS_PORT
TENCENTCLOUD_REDIS_PASSWORD
```

### 7.2 需要手动配置的变量

云托管控制台 -> **服务详情** -> **环境变量** -> **添加变量**

| 变量名 | 值 | 必填 |
|--------|-----|------|
| `APP_VERSION` | `1.0.0` | ✅ |
| `DEBUG` | `false` | ✅ |
| `PORT` | `8080` | ✅ |
| `JWT_SECRET_KEY` | `<生成随机密钥>` | ✅ |
| `WECHAT_APP_ID` | `wx8888888888888888` | ✅ |
| `WECHAT_APP_SECRET` | `<小程序密钥>` | ✅ |
| `OCR_APP_ID` | `<百度 OCR AppID>` | ⏭️ |
| `OCR_API_KEY` | `<百度 OCR API Key>` | ⏭️ |
| `OCR_SECRET_KEY` | `<百度 OCR 密钥>` | ⏭️ |

> 💡 JWT_SECRET_KEY 生成方法：
> ```bash
> # Linux/Mac
> openssl rand -hex 32
> 
> # Windows PowerShell
> [System.BitConverter]::ToString((New-Object Security.Cryptography.RNGCryptoServiceProvider).GetBytes(32)).Replace("-","")
> ```

## 8. 配置小程序域名

### 步骤 1：获取云托管服务域名

1. 云托管控制台 -> **服务管理**
2. 复制服务域名
   - 格式：`https://<服务 ID>-<环境 ID>.service.tcloudbase.com`

### 步骤 2：配置合法域名

1. 微信公众平台 -> **开发** -> **开发设置**
2. 服务器域名配置
3. 点击「修改」
4. 添加以下域名：

```
request 合法域名：
  https://<云托管服务域名>

uploadFile 合法域名：
  https://<云托管服务域名>

downloadFile 合法域名：
  https://<云托管服务域名>
```

> ⚠️ 注意：
> - 必须使用 HTTPS
> - 域名不能带端口号
> - 每月只能修改 5 次

## 10. 故障排查

### 问题 1：数据库连接失败

**症状：** 500 错误，日志显示 "Connection refused"

**解决方案：**
1. 检查云托管和数据库是否在同一环境
2. 确认环境变量已正确配置
3. 检查数据库白名单是否允许云托管访问

### 问题 2：冷启动慢

**症状：** 首次请求需要 5-10 秒

**解决方案：**
1. 设置最小实例数为 1-2
2. 使用「常驻实例」功能
3. 优化代码启动速度

### 问题 3：CORS 错误

**症状：** 小程序请求失败，显示 "CORS policy"

**解决方案：**
1. 检查 `backend/main.py` 中的 CORS 配置
2. 确保 `allow_origins` 包含小程序域名
3. 生产环境不要使用 `["*"]`

### 问题 4：环境变量未生效

**症状：** 应用使用默认配置

**解决方案：**
1. 重启服务（云托管控制台 -> 服务详情 -> 重启）
2. 检查变量名是否正确（区分大小写）
3. 确认变量已保存
